# Cloud-Computing-Project-Eurovision-Rankings
For my Spring 2025 Cloud Computing course, we had to do a project where we had to make a web application of a topic of your choice. As a Eurovision fan, I chose to do this over my Eurovision rankings.

All of the files provided are all the original files from when I presented the project in May. None of these files have been touched.

This project was done using Google Cloud and Google App Engine. All of the HTML files are downloaded from my spreadsheets consisting of my rankings. These rankings include my actual ESC Rankings, National Final Rankings, Other non-ESC rankings, and one ideal ranking for 2026. Since this project was so long ago and it was a simple thing, it is not the best and there are some things I do wish to improve on at a later date.

I plan to come back to this as a side project at some point when I get better at developing these web applications and understand HTML and CSS well. A major change I wish to do is utilizing Google Sheets code for dynamic site changes instead of needing to redownload and reupload html files, especially for national finals.

Another plan is to input a separate section for the upcoming ESC Asia.

If you wish to view my current updated rankings and what national finals I've ranked (as well as my other related rankings), there is a link to each spreadsheet towards the bottom of each html in the "templates" folder except home.html. It is within one of the <a>, same line as "The Master Spreadsheet" for each
